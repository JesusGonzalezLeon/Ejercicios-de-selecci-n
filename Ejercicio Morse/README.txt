Informacion del programa
El codigo del programa se encuentra en el archico CodificadorMorse.java y se puede ejecutar el programa dede la cosola direccionando la ubicacion de los archivos y con ayuda de las heramientas de java jvm y jre se utilizan los comandos (javac CodificadorMorse.java y java CodificadorMOrse). 
El programa pedira que se ingrese un mensaje, el cual se codificara o decodificara acorde a lo ingresado si el mensaje contiene letras y numero seran convertidos a codigo morse de tener codigo morse sera convertido a texto, en caso de contener la combinacion de ambos o caracteres inadecuados el programa se lo hara saber.
