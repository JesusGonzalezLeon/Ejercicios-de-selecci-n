Informacion del programa
El codigo del programa se encuentra dividido en 4 archivos en en el archico Cuadrado.java y Cubo.java se encuentra el codigo de las clases donde se puede ver el uso de la herencia. En los archivos Pricipal.java y Menu.java se encuentra el condigo de controlador para el programa.
Se puede ejecutar el programa dede la cosola direccionando la ubicacion de los archivos y con ayuda de las heramientas de java jvm y jre se utilizan los comandos 
1.- javac Principal.java Cuadrado.java Cubo.java Menu.java
2.- java Principal Cuadrado Cubo Menu
El programa solicita un numero que sera el tamaño de la figura y despues mostrara las opciones de uso del programa en un meno el cual se respondera con numeros referentes a cada posible operacion.
